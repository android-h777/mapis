APIS MPM Chem Studio — 프로젝트 소개 스크립트
================================================


1. 오프닝
---------

안녕하세요. 오늘 소개해드릴 것은 APIS MPM Chem Studio입니다.
KCC World APIS 시스템의 신규 모듈로, 화학 분자 구조를 웹 브라우저에서 직접 그리고 관리할 수 있는 화학 구조식 에디터입니다.
기존에는 별도의 데스크탑 소프트웨어(ChemDraw, MarvinSketch 등)에 의존했던 분자 구조 작업을, APIS 플랫폼 안에서 원스톱으로 처리할 수 있도록 설계했습니다.


2. 해결하고자 하는 문제
-----------------------

신제품 개발(NPDI) 과정에서 화학 구조식은 핵심 정보입니다.
그러나 기존 프로세스에서는:
- 구조식을 외부 프로그램에서 그린 뒤 이미지로 캡처해서 첨부
- 데이터로 관리되지 않아 검색, 비교, 재활용이 불가능
- 라이선스 비용이 높은 상용 소프트웨어에 의존

Chem Studio는 이 문제를 해결합니다. 웹 브라우저만 있으면 누구나, 어디서든 분자 구조를 그리고 APIS 시스템에 바로 연결할 수 있습니다.


3. 주요 기능
------------

3-1. 분자 카드 리스트

첫 화면은 등록된 분자 구조들의 카드형 리스트입니다.
각 카드에는 분자의 2D 구조식 SVG, 분자식(Molecular Formula), 등록자, 등록일이 표시됩니다.
카드에 마우스를 올리면 물리 기반 3D 틸트 효과가 적용됩니다 — Schlick의 프레넬 근사(Fresnel Approximation)를 적용해 실제 유리 표면의 반사와 색수차(Chromatic Aberration)까지 재현했습니다.
카드 진입 시 flipInX 애니메이션으로 순차 등장하며, 시각적으로 몰입감 있는 경험을 제공합니다.


3-2. 2D 분자 에디터

카드를 클릭하거나 "New Editor" 버튼을 누르면 전체 화면 모달로 2D 에디터가 열립니다.

[도구 모음(Tools Area)]
- 원소 슬롯: C, H, O, N, S, P, F, Cl, Br, I 등 자주 쓰는 10개 원소가 기본 배치
- 드래그 앤 드롭으로 원소 순서를 자유롭게 변경 가능
- 주기율표 팝업: '+' 버튼을 누르면 전체 118개 원소가 담긴 인터랙티브 주기율표가 나타남
- 결합 도구: 단일결합, 이중결합, 삼중결합
- 구조 템플릿: 오각형, 육각형, 벤젠 고리, 원형 라디칼
- 폰트 선택, Undo/Redo, 그리드 토글

도구 모음 위에 마우스를 움직이면 스페큘러 하이라이트가 따라다니며, 도구 영역 자체에도 유리 질감을 느낄 수 있습니다.


3-3. 인터랙티브 주기율표

주기율표는 단순한 선택 UI가 아닙니다.
- 118개 원소가 랜덤 팝인 애니메이션으로 등장
- 원소 카테고리별 색상 코딩 (알칼리 금속, 전이 금속, 비금속, 할로겐, 희가스 등 11종)
- 역제곱 법칙(Inverse-square law) 기반 근접 글로우 — 마우스에 가까운 원소일수록 밝게 빛남
- 각 원소에 원자 번호, 기호, 질량, 전기음성도(Pauling) 표시
- 방사성 원소는 붕괴 애니메이션으로 구분
- 이미 도구 모음에 추가된 원소는 체크 표시
- 마우스 호버 시 실시간 커스텀 툴팁으로 상세 정보 표시


3-4. SMILES/Formula 통합 검색

에디터 하단에 통합 검색바가 있어 Formula, 화합물 명칭, CAS 번호로 검색할 수 있습니다.
AI 채팅 패널이 열려있을 때는 검색바가 자동으로 숨겨지고, 채팅 패널을 닫으면 다시 나타나는 등 컨텍스트에 맞는 UI 전환을 제공합니다.


3-5. AI Assistant (채팅 패널)

에디터 우측에 AI 어시스턴트 채팅 패널이 내장되어 있습니다.
- "벤젠 그려줘", "OH기 추가해줘" 같은 자연어 명령으로 분자 구조를 제어
- 글라스모피즘(Glassmorphism) 디자인 — backdrop-filter: blur(40px) 적용
- 패널 뒤로 RGB 오오라가 흐르며 AI 기능임을 시각적으로 강조
- 열기/닫기 시 버튼에서 패널로 변환되는 모프(Morph) 애니메이션 + 스프링 바운스 효과
- 내부 콘텐츠(헤더, 메시지, 입력창)는 시차 페이드인으로 순차 등장
- 타이핑 인디케이터(점 3개 애니메이션)로 AI 응답 대기 상태 표현


3-6. 2D View / 3D View 탭

상단 Apple 스타일 탭으로 2D Editor, 2D View, 3D View 모드를 전환할 수 있습니다.
슬라이딩 인디케이터가 탭 이동에 따라 부드럽게 이동합니다.


4. 디자인 철학
--------------

Chem Studio의 디자인은 세 가지 원칙을 따릅니다:

1) 과학적 정밀함
   원소 데이터, 전기음성도, 원자 질량 등 실제 화학 데이터를 충실히 반영

2) 물리 기반 시각 효과
   프레넬 반사, 역제곱 법칙 글로우, 스프링 바운스 등 물리 법칙에 기반한 자연스러운 인터랙션

3) 글라스모피즘 + 미니멀
   반투명 유리 질감, 부드러운 그라데이션, 절제된 색상 사용으로 화학 데이터가 돋보이는 깔끔한 UI


5. 기술 스택
------------

프론트엔드      순수 HTML/CSS/JavaScript (빌드 도구 없음)
UI 프레임워크   Materialize CSS v1.0.0, Material Icons
DOM 조작        jQuery 3.3.1
분자 구조 렌더링  SVG (인라인, 순수 JavaScript 생성)
애니메이션      CSS Transitions/Animations + animate.css
백엔드 연동     Java REST API (apis.kccworld.co.kr)
폰트            SUIT (Variable), Noto Sans KR, Roboto

별도 빌드 시스템 없이 HTML 파일을 직접 서빙하는 구조로, 배포가 간단하고 유지보수가 용이합니다.


6. APIS 시스템과의 연계
-----------------------

Chem Studio는 독립 모듈이 아니라 APIS MPM(Material & Product Management) 시스템의 일부입니다.
- NPDI(신제품 개발): 신제품 개발 프로세스에서 분자 구조를 직접 등록하고 관리
- 기술 문서: 구조식을 기술 문서에 바로 삽입
- FMEA: 소재 관련 고장 모드 분석 시 구조식 참조
- 제안/승인: 제안서에 분자 구조 정보를 포함하여 의사결정 지원

하나의 플랫폼 안에서 분자 설계부터 제품 개발까지 끊김 없는 워크플로우를 구현합니다.


7. 향후 계획
------------

- AI 연동 고도화: 자연어로 복잡한 분자 합성 경로 제안, 유사 구조 검색
- 3D 분자 뷰어: WebGL 기반 3차원 분자 회전/확대 뷰어
- SMILES 파서: SMILES 문자열 입력 → 자동 2D 구조 생성
- 구조식 DB 검색: CAS 번호, InChI Key 기반 외부 데이터베이스 연동
- 협업 기능: 여러 연구원이 동시에 같은 구조식을 편집


8. 클로징
---------

Chem Studio는 화학 연구원이 별도 소프트웨어 없이 웹에서 바로 분자 구조를 그리고, AI의 도움을 받아 더 빠르게 작업하며, APIS 시스템과 자연스럽게 연결되는 경험을 제공합니다.
감사합니다.
